# StarMade Guides

Welcome to the [StarMade](https://www.star-made.org/) Guides! This is a replica of the in-game documentation [built by the community](https://github.com/StarMade-Community/StarMade-Guides) to help players get started in StarMade. The *old* documentation can be found [here](https://starmadepedia.net/wiki/StarMade_Wiki), but is not maintained. For the most up to date information please refer to this site or to the in-game guides. Click on the [Introduction](00-getting-started/00-introduction.md) to get started.

## Site Navigation

Browse the navigation on the left to find guides on various topics. Topics are broken out into sections of general gameplay mechanics and you can search for keywords as needed in the top left searchbar.
